# 🦠 COVID-19 Health Analytics Dashboard

A beautiful, interactive COVID-19 data analytics dashboard built with Python and Streamlit.

## Features
- 📈 Time series trends (cases, deaths, active, vaccination)
- 🗺️ Country-wise comparison charts
- 🗓️ Monthly heatmap
- 💊 Recovery rate analysis
- 📋 Summary data table
- 🔍 AI-generated insights
- 🎛️ Interactive filters (countries, years, metrics)
